# DLME Website Archive

This is an archive of the web pages that lived at http://dlme.clir.org during
the initial phases of the DLME project. They include early promotional
information and content related to the Whiting and Mellon phases of the work. An
archive of this content can be found on
[the WaybackMachine](https://web.archive.org/web/*/https://dlme.clir.org).

## Local Usage

* Clone the repository
* Run an http server of your choice in the `docs` directory

  git clone https://github.com/clirdlf/dlme.clir.org.git
  cd dlme.clir.org/docs
  php -S localhost:8000
